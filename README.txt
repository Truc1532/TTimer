---DEPENDENCIES---
  make
  C headers:
  stdio.h, string.h, stdlib.h, unistd.h, termios.h, stdint.h

--INSTALLATION---
  First run
  chmod +x autogen.sh
  Then run the shell script
  ./autogen.sh
  After that's done run the configure script
  ./configure
  Once configured build the program
  make
  And Finally to install the program on you system run
  sudo make install
